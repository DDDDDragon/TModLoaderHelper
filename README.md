# TModLoaderHelper使用说明

- 编辑.cs文件时输入Item+物品中文名可以自动变为ItemID 如Item铁镐 => ItemID.IronPickaxe
- 悬停在物品英文名称上可以自动显示对应的中文名
- 插件会自动读取
## 版本说明
